# КалориСкан — подсчёт калорий по фото

## Установка

```powershell
pip install -r requirements.txt
```

## Запуск

```powershell
$env:ANTHROPIC_API_KEY = "ваш-api-ключ"
python server.py
```

Откройте браузер: http://localhost:5000

## Использование

1. Загрузите фото еды (перетащите или нажмите «Выбрать фото»)
2. Нажмите «Анализировать калории»
3. ИИ определит блюда и рассчитает КБЖУ
